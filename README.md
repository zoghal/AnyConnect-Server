# AnyConnect-Server


Installation Script for Cisco AnyConnect

Linux:
- centos
- ubuntu

## Setup

```bash
wget -N --no-check-certificate https://raw.githubusercontent.com/zoghal/AnyConnect-Server/main/ocserv-en.sh
chmod +x ocserv-en.sh
bash ocserv-en.sh
```
